Beautiful Resume CV Gutenberg Block for everyone who wants to publish a nice Resume/CV.

== Description ==

Clean, Modern and Professional Resume CV Gutenberg Block designed to make an impression.

Very Easy to edit and customize the Resume/CV, with 4 Gutenberg blocks included. All blocks can be arranged in a nice order to perfectly build your Resume/CV. This is a very good solution for everyone who wants to publish the Resume CV on their WordPress Website.

== Installation ==

You have to search for the "**Resume CV Block**" via the **Plugins -> Add New** dialog and click install, or download and extract the plugin, and copy the plugin folder into your wp-content/plugins directory and activate.

After installation create a new page/post using the Gutenberg editor, and when adding a new block, look under  **Resume Blocks** or type **Resume**.

- Select the Resume Template Block
- Inside of Template Block, Select the Resume Sections Block
- Each section will allow you to select different Blocks like List, Paragraph, Education or Experience block
